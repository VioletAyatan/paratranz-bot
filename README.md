# ParaTranz-bot

一个简单的基于`ParaTranz`网站`web-api`构建的qq机器人应用。

## 功能

1. 项目申请提醒（项目有新人申请加入时，自动提醒指定群组）
2. 针对`paraTranz网站Api`的指令集成。
3. 更多功能开发中（鸽:dove:）

## 技术栈

`qq-bot`实现，`mirai`，参考：https://github.com/mamoe/mirai

## 指令集

- `/audit` 自动将申请项目的新人翻译内容转发到群组中。
- `/help` 获取指令帮助信息。
- Work in progress...
